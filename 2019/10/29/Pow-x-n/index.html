实现 pow(x, n) ，即计算 x 的 n 次幂函数。

示例 1:

输入: 2.00000, 10
输出: 1024.00000

说明:

  • -100.0 < x < 100.0
  • n 是 32 位有符号整数，其数值范围是 [−2^31, 2^31 − 1] 。
